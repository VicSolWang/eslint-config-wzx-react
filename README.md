The custom react eslint rules based on airbnb

# Installation

    npm install --save-dev eslint-config-wzx-react

# Usage

Add eslint-config-wzx-react to the "extends" in your .eslintrc.* file.

    {
        "extends": "eslint-config-wzx-react"
    }
